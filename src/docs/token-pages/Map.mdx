# Map tokens

The **map** tier derives from seed via `color-mix()` and scale math — 10-state
colour ramps, sizing, radius and type scales. The **Value** column shows the
derivation (`var(--seed-…)` / `color-mix(…)`); **Resolved** is the live browser
value under the current Heat / Depth / Brand.
