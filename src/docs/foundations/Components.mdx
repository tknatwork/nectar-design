# Component Gallery

A survey of every component the package exports — all variants
side-by-side so the system reads as one. Every component below imports
from `nectar-design` and renders against the engine canvas, so flipping
the toolbar **Heat × Depth** repaints them live.

For interactive component playgrounds (Dialog, Popover, Toast, Menu,
Tabs, Tooltip), see their individual stories in the sidebar.

## What's not on this page

Stateful components require a story sandbox to render their open / closed
states meaningfully — they live in the **Components/** sidebar group:

- `Dialog` — modal with backdrop + focus trap
- `Popover` — anchored floating content
- `Tooltip` — hover / focus reveal
- `Menu` — keyboard-navigable command list
- `Tabs` — segmented panel switcher
- `Toast` — ephemeral notifications via `toast()` imperative API
- `Select` — combobox with keyboard nav
- `Checkbox` / `Radio` / `Switch` — form controls (variants here, behavior in stories)
- `Breadcrumbs` / `Table` / `List` — composition components
