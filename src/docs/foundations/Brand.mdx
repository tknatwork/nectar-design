# Brand

The verbal + visual identity that's *not* engine-driven — wordmark, favicon,
voice, and the static ink palette used by Ink Routes. Anything you see here
holds steady when the toolbar Heat / Depth flips.

## Wordmark

The signature is **Tushar Kant** in Plus Jakarta Sans tracked tight, plus
**Naik** in Playfair italic accent. The italic surname gets `--accent`
color so it breathes even though the rest of the wordmark is static.

## Favicons

Three sizes generated from the same lowercase **t** glyph on a radial
accent gradient. Browsers pick the closest size; PWA + Apple Touch use the
larger asset with a deeper background.

## Identity tints

Per-project glass tints via `WORK_SECTION_OVERRIDES.glassStyle` (see
`Patterns/Overview` § CaseStudyTile). Each tint mixes a fixed identity
hue with the live `--dynamic-hue` so it still breathes with Heat.
