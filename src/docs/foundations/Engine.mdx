# Engine — Heat × Depth in real time

This page is painted by the runtime engine that drives the live portfolio
at tusharkantnaik.com. Two CSS variables, `--ui-heat` and `--ui-depth`,
feed an oklch expression that recomputes every paint-relevant token
(`--bg`, `--fg`, `--primary`, `--surface`, `--muted-fg`, `--border`,
`--ring`) on every frame.

**Try it:** use the **Heat** and **Depth** toolbar dropdowns above to
sweep the engine. Both axes update through a 450ms ease-in-out
animation; depth additionally fires a Gaussian-shaped golden-hour hue
burst (peaks at depth=40), echoing the live site's theme transition.

## How the formula works

```css
:root {
  --ui-heat: 0;     /* 0–100 → hue (cool blue → warm amber) */
  --ui-depth: 100;  /* 0–100 → lightness (light → dark) */

  /* The lynchpin scalar — every oklch in the system reads this hue */
  --dynamic-hue: calc(250 - (var(--ui-heat) * 2.1) + var(--golden-hue-shift));

  /* Lightness anchors interpolate by depth */
  --L-bg:      calc(0.96 - (var(--ui-depth) / 100) * 0.81);
  --L-heading: calc(0.20 + (var(--ui-depth) / 100) * 0.75);
  --L-body:    calc(0.35 + (var(--ui-depth) / 100) * 0.33);
  --L-muted:   calc(0.50 + (var(--ui-depth) / 100) * 0.10);
  --L-accent:  calc(0.50 + (var(--ui-depth) / 100) * 0.22);
  --L-surface: calc(0.98 - (var(--ui-depth) / 100) * 0.80);

  /* Chroma is depth-interpolated + transient golden-hour boost */
  --C-bg:     calc(0.02 + (var(--ui-depth) / 100) * 0.06 + var(--golden-chroma-boost));
  --C-accent: calc(0.18 - (var(--ui-depth) / 100) * 0.04 + var(--golden-chroma-boost));

  /* Every semantic var is `oklch(L C H)` of the above */
  --bg:      oklch(var(--L-bg)      var(--C-bg)     var(--dynamic-hue));
  --fg:      oklch(var(--L-body)    0.03            var(--dynamic-hue));
  --surface: oklch(var(--L-surface) 0.02            var(--dynamic-hue));
  --primary: oklch(var(--L-accent)  var(--C-accent) var(--dynamic-hue));
  /* …and so on */
}
```

## The golden-hour transition

When you flip Depth (light ↔ dark), the engine doesn't just interpolate
luminance — it sweeps through a warm-amber midpoint, mimicking the way
real light shifts at sunrise/sunset. Implementation:

```ts
const goldenStrength = Math.exp(-((depth - 40) ** 2) / (2 * 20 ** 2)); // gaussian, σ=20
root.style.setProperty('--golden-hue-shift', goldenStrength * -180);   // shift hue toward warm
root.style.setProperty('--golden-chroma-boost', goldenStrength * 0.12); // saturate during peak
```

The bell curve peaks at depth=40 and fades to ~0 at the extremes —
visitors see a *sunset* between light and dark, not just a luminance ramp.
