## Anatomy

A breadcrumb is an ordered list inside a labelled `nav`. Each `items` entry is a link except
the last, which renders as plain `current-page` text; an `aria-hidden` chevron separator sits
between entries. Pass `separator` to swap the glyph.

## Usage

- **Do** — mirror the real navigation hierarchy and make every ancestor except the last a working link.
- **Don't** — use a breadcrumb as the primary nav, or list steps in a flow (it reflects *location*, not *history*).

## Accessibility

- Wrapped in `<nav aria-label="Breadcrumb">` so assistive tech announces it as the breadcrumb landmark.
- The final crumb carries `aria-current="page"` and is the only non-link entry.
- Separators are `aria-hidden="true"` — decorative, never read aloud.
- Use the **Accessibility** tab on any story canvas for the live axe audit.
