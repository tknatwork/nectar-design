## Anatomy

A square control with a 2px border that fills with `primary` when checked, showing a
check glyph — or a minus glyph when `indeterminate`. Built on Base UI for state and
focus management; the box scales with `size` while the glyph stays centred.

## Usage

- **Do** — use a checkbox for an independent on/off choice (one box) or several non-exclusive options stacked together.
- **Don't** — use checkboxes for mutually exclusive choices; that's a `RadioGroup`. Reserve `indeterminate` for a parent summarising mixed children.

## Accessibility

- Give every checkbox a name: a wrapping `<label>` / `<Label htmlFor>`, or `aria-label` when standalone.
- `indeterminate` is a visual + `aria` mixed state — it is not a third value; the box is still semantically checked or not.
- Fully keyboard-operable (Space toggles); `disabled` removes it from the tab order via Base UI.
- Use the **Accessibility** tab on any story canvas for the live axe audit.
