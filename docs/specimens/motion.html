Motion

Four easing curves, a six-step duration scale, three Framer Motion spring presets, and the long colour cross-fade. Numbers are taken verbatim from colors_and_type.css and GlassCard.tsx — drag Heat at the bottom and the whole page re-paints over --dur-cross (800 ms).

Easing

Default is --ease-out. The ease-out-circ variant is reserved for hero-scale motion (full-bleed entries, page transitions).

--ease-outcubic-bezier(0.215, 0.61, 0.355, 1)
--ease-out-circcubic-bezier(0.08, 0.82, 0.17, 1)
ease-in-outcubic-bezier(0.65, 0, 0.35, 1)
linearfallback only

Duration

Most UI motion sits at --dur-base (200 ms). The cross-fade is intentionally long because it's ambient, not interactive.

--dur-fast
120 ms
Hover, focus.
--dur-base
200 ms
Default for buttons, tabs.
--dur-slow
300 ms
Card lift, modal open.
--dur-cross
800 ms
Theme + heat colour cross-fade.
circadian
60 s
Solar-driven type / shadow drift. Ambient — never tied to interaction.

Springs

Production uses Framer Motion springs. The CSS approximations below match the feel — pop, settle, and overshoot — to within a few ms. Click any cell.

--spring-nav

stiffness 200 · damping 30

GlassNav active pill slide

Cross-fade

All colour-bearing properties (background-color, color, border-color) carry the 800 ms transition by default — so theme and heat changes feel like the page is breathing, not snapping.

Reduce motion

Production listens to prefers-reduced-motion and drops every transition to 0 ms. Toggle the override below to feel the difference.
